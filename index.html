ALUX framework 1.0

The Semantic Framework for Awesome Light User Experience

ALUX is a Superlight framework (less than 10K) for create amazing responsive application.
The framework is buildt with HTML/CSS and pure Javascript. Grid, Animated Navigation, Typography, Forms, Buttons & more.

Easy, Light, Customizable. Example: One Page Structure

DOWNLOAD GIT HUB

01. Multiple Grids

The grid sistem use fractions to separate the width of grid in columns. For example 1/2 of the width is .col-1-2. You can set 2/3/4/5/8 Columns Grid.

1/2 .col-1-2
1/2 .col-1-2
1/3 .col-1-3
2/3 .col-2-3
1/4 .col-1-4
3/4 .col-3-4
1/5 .col-1-5
4/5 .col-4-5
1/8 .col-1-8
7/8 .col-7-8 
<div class="grid">
  <div class="col-1-2">
        1/2
  </div>
  <div class="col-1-2">
       1/2
  </div>
</div>

02. Semantic Structure

Add classes to HTML5 Tag for light and semantic markup.

2/3 article .col-2-3
1/3 sidebar .col-1-3 
<div class="grid">
    <article class="col-2-3">
        2/3 article 
    </article>
    <aside class="col-1-3">
       1/3 sidebar 
    </aside>
</div>

05. Push Columns

You can push a column in every base grid. The push system works on Tablet and Desktop.

3/4 .col-3-4 push-1-4
5/8 .col-5-8 push-3-8
<div class="grid">
  <div class="col-3-4 push-1-4">
    3/4
  </div>
</div>

06. Push & Pull Columns

You can push & pull columns in every base grid. The pull system works on Tablet and Desktop.

Content .col-3-4 push-1-4
Sidebar .col-1-4 pull-3-4
<div class="grid">
  <div class="col-3-4 push-1-4">
    Content
  </div>
  <div class="col-1-4 pull-3-4">
    Sidebar
  </div>
</div>

07. Mobile Options (try resize the Screen)

The colums on Smartphone are 100% of the width. The colums on Tablet are the same of the desktop.
You can overwrite the default width of a columns in Smartphone & Tablet view with the classes .sma-% and .tab-% (% => 25, 33, 50, 66, 100).
For example: tab-33 sma-50

1/4 .col-1-4 tab-100
1/4 .col-1-4 tab-100
1/4 .col-1-4 tab-50 sma-50
1/4 .col-1-4 tab-50 sma-50
<div class="grid">
  <div class="col-1-4 tab-100">
    1/4
  </div>
  <div class="col-1-4 tab-100">
    1/4
  </div>
  <div class="col-1-4 tab-50 sma-50">
    1/4
  </div>
  <div class="col-1-4 tab-50 sma-50">
    1/4
  </div>
</div>

08. Push Mobile (try resize the Screen)

You can overwrite the normal push on Tablet with the class .tab-push-%, on Smartphone with the class .sma-push-% . (% => 25, 33, 50, 66, 100).
For example: tab-push-33 sma-push-50
